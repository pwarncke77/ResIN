# ResIN

Response-Item Networks (ResIN) Package for R

Maintainer: Philip Warncke <pwarncke@live.unc.edu>

Authors: Philip Warncke, Dino Carpentras, and Adrian Lüders

Description: Contains various tools to estimate and visualize Response
Item Networks (ResIN’s). ResIN binarizes ordered-categorical and
qualitative response choices from (survey) data, calculates pairwise
associations and maps the location of each item response as a node in a
force-directed network. Please refer to <https://www.resinmethod.net/>
for more details.
